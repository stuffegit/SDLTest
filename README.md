# SDL test project

- Static 1920x1080 window(Esc to kill).
- WASD movement.
- Camera-respecting fog.
- Simple meshes with hardcoded colours.
- Debug menu(ImGui) that saves settings(JSON).

![Funnyimage](/readme/24may_2026.jpg "Badass")

### Build
Windows
```
.\build.bat && .\build\debug\SDLTest.exe
```
Linux
```
./build.sh 
./build/debug/SDLTest
```
